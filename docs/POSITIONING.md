# 定位话术（Positioning One-liner）草稿

> 本文件为「岸灯鸢花」对外定位标语的草稿，供 README 顶部引入区与发布物料复用。
> 文案可迭代，最终对外口径以团队确认版本为准。

## 当前推荐标语

> 一个本地优先的轻量保险库，把你的阅读、图片、视频、音乐与笔记收纳进同一个私密空间——数据自主、离线可用、插件可热插拔。

## 关键要素拆解

| 要素 | 说明 | 在标语中的体现 |
| --- | --- | --- |
| 本地优先 | 数据默认存于本机，不强制上云 | 「本地优先」「离线可用」 |
| 五模态聚合 | 阅读 / 图片 / 视频 / 音乐 / 笔记 统一收纳 | 「阅读、图片、视频、音乐与笔记」「同一个私密空间」 |
| 数据自主 / 隐私 | 用户掌控自己的文件与元数据 | 「私密空间」「数据自主」 |
| 可热插拔 | 插件运行时动态加载、增删不重启 | 「插件可热插拔」 |

## 备选方向（未采用）

- 「一个本地优先的轻量保险库，把你的阅读、图片、视频、音乐与笔记装进同一个私密空间——数据自主、离线可用、插件可热插拔。」（偏「装进」动作感）
- 「把阅读、图片、视频、音乐与笔记收进同一个离线保险库：本地优先、数据自主、插件热插拔。」（更短，信息密度高）

## 使用约定

- README 顶部引入区使用「当前推荐标语」作为主文案。
- 对外物料（发布说明、商店简介、社交简介）优先复用同一句，保持口径一致。
- 如产品定位发生调整，请同步更新 `README.md` 与本文档。
