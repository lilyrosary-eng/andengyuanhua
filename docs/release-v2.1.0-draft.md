# 岸灯鸢花 v2.1.0-alpha.1 发布草稿

> ⚠️ 状态：**alpha 草稿**，请勿直接对外宣布。
> 本文档用于内部对齐首个公开发布的范围与措辞，发布前需补充真实链接、校验和与最终确认。

## 版本

- Release 标签：**v2.1.0-alpha.1**（首个 alpha 通道；产品内部版本仍为 2.1.0）
- 安装包示例名：`岸灯鸢花_2.1.0_x64-setup.exe`（由 `src-tauri/tauri.conf.json` 的 `version=2.1.0` 生成）
- 通道：**alpha**（首个公开版本，稳定性与覆盖面仍在打磨，请勿用于生产关键数据）

## 定位

> 一个本地优先的轻量保险库，把你的阅读、图片、视频、音乐与笔记收纳进同一个私密空间——数据自主、离线可用、插件可热插拔。

## 已知范围

- **平台**：仅 **Windows 10 / 11**（当前构建绑定 Windows 原生能力：全局截图、录屏、原生拖出等）。
- **跨平台**：Linux / macOS 已有设计规划，尚未落地，后续版本推进。
- **稳定性**：首个公开版本，可能存在已知但未穷尽的问题；建议在使用前备份个人数据。

## 核心功能清单

- 笔记核心：富文本编辑（TipTap）、全文搜索、标签、置顶、快照存档与恢复、整库备份导出、浮窗笔记。
- 插件系统：`plugin://` 私有协议动态加载，IIFE 沙箱隔离，运行时热插拔；内置与第三方用户插件并存（用户插件可覆盖内置）。
- 音乐 · 铃兰：本地音乐扫描与播放、元数据/歌词解析、可锁定桌面悬浮歌词窗。
- 图片 · 莲花：图库扫描、缩略图生成与缓存。
- 视频 · 玉兰：视频库扫描与文件夹浏览。
- 阅读 · 三色堇：EPUB 等电子书解析阅读。
- 办公 · 茑萝：docx / pptx 导入导出、CodeMirror IDE 与绘画子模块。
- 专业 · 薄荷：环境变量、端口扫描、进程列表、剪贴板、格式转换、ffmpeg 转码等工具集合。
- 全局截图 / 录屏：微信式全局截图（多屏、点窗、长截图、标注）、屏幕录制。
- 系统集成：单实例守卫、系统托盘、跨模块「中转站」暂存区、会话日志。
