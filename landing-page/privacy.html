隐私政策

最后更新：2026 年 7 月 19 日

安得云荟（以下简称“本应用”）是一款本地优先的桌面工具，由 React + Rust（Tauri）构建。我们深知隐私的重要性，因此本应用在设计之初就遵循 “数据自主、隐私为本” 的核心原则。

本隐私政策旨在帮助您了解本应用如何处理您的信息。请在使用本应用前仔细阅读。

核心承诺： 本应用 不上传 您的任何个人数据至云端。所有笔记、插件数据、媒体库信息、截图、录屏等，默认全部存储在您本地设备上，由您完全掌控。

1. 我们收集的信息

1.1 主动提供的信息

当您使用本应用时，您可以主动创建或导入以下数据：

  • 笔记内容：您编写的富文本笔记、标签、分类等。
  • 媒体文件：您导入的本地音乐、图片、视频、电子书等。
  • 文档数据：您编辑或导入的办公文档（docx、pptx、xlsx 等）。
  • 截图与录屏：您通过全局截图或录屏功能生成的图像/视频文件。
  • 插件配置：您安装或配置的插件相关设置。

所有这些数据 仅保存在您的本地磁盘，本应用不会主动收集或上传这些内容。

1.2 自动收集的信息

本应用 不会 自动收集任何个人信息，包括但不限于：

  • 设备唯一标识符（如 MAC 地址、硬盘序列号）
  • IP 地址
  • 地理位置
  • 浏览历史或使用习惯
  • 崩溃报告或使用统计数据

本应用 不包含任何跟踪器、分析 SDK 或第三方统计工具。

2. 数据的存储与安全

2.1 存储位置

  • 所有用户数据默认存储在您的本地应用数据目录中（例如 Windows 下的 /Downloads/安得云荟）。
  • 您可以通过应用内的“备份”功能将数据导出到您指定的位置。
  • 您也可以手动删除应用数据目录来清除所有数据。

2.2 数据传输

本应用 不会 通过网络传输您的任何数据。所有操作均在本地完成。

2.3 安全措施

  • 本应用不涉及网络通信，从根本上避免了数据在传输过程中被截获的风险。
  • 应用本身遵循 Rust 和 Tauri 的安全最佳实践，包括沙箱隔离、最小权限原则等。
  • 您导入的插件在独立的 IIFE 沙箱中运行，受限的 API 访问确保插件无法越权访问您的系统或数据。

3. 第三方服务

本应用 暂不依赖任何第三方云服务 来存储或处理您的数据。所有功能在离线状态下完整可用。

应用内置的插件系统允许您加载第三方插件，但这些插件同样运行在本地沙箱中，其隐私行为由插件开发者自行声明。建议您仅从可信来源安装插件。

4. 您的权利

由于所有数据均存储在本地，您对您的数据拥有 完全的控制权：

  • 查阅：您可以随时通过应用界面查看您的笔记、媒体库等所有数据。
  • 修改：您可以随时编辑或更新您的数据。
  • 删除：您可以随时删除单个条目或通过清除应用数据目录来删除全部数据。
  • 导出：您可以通过内置的备份功能将数据导出到您指定的位置。

5. 儿童隐私

本应用不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。如果您是家长或监护人，并且认为您的孩子向我们提供了个人信息，请通过下方联系方式与我们联系。

6. 隐私政策的变更

我们可能会不时更新本隐私政策。我们会在本页面顶部更新“最后更新”日期。建议您定期查看本页面以了解任何变化。

由于本应用不联网，我们无法通过应用内推送通知告知您变更，因此请您主动关注应用发布页或 GitHub 仓库的公告。

7. 联系我们

如果您对本隐私政策有任何疑问、建议或担忧，请通过以下方式联系我们：

  • 在 GitHub Issues 中提交反馈：github.com/lilyrosary-eng/andengyuanhua/issues

✅ 总结： 安得云荟是一款 完全本地、离线的桌面应用。我们不会收集、上传或分享您的任何数据。您的数据只属于您自己。
